HTML Tag Glossary

This website is a giant list of html tags and how to use them. The tags are categorized into groups that describe what the tag is used for.

Boiler Plate Tags

These tags are the most common tags that appear in almost every html document. These tags are considered to be apart of the starter code for any html document.

<!DOCTYPE html>

This tag tells the browser what kind of html document the page is. This tag is placed at the top of every HTML document.

<html>

This tag is the starting point and end point for any html document. The html tag encloses all tags in an html document.

<head>

This tag contents all of the meta data for an html document. Meta data is information about the page but not information that shows visually on the page.

<meta charset="UTF-8">

This tag contents all of the meta data for an html document. Meta data is information about the page but not information that shows visually on the page. Most browsers use UTF-8 by default if no character encoding is specified but because that is not guaranteed, its better to include a character encoding specification using the <meta> tag in your HTML file.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tag gives the browser instructions on how to control the pages's dimensions and scaling. The name=viewport statement tells the browser that the viewport is going to be edited. The content artibute has two major statements. The width=device-width statements sets the width of the page to follow the screen-width of the device(which will vary depending on the device). The initial-scale=1.0 statement sets the initial zoom level when the page is first loaded by the browser.

<meta http-equiv="X-UA-Compatible" content="IE=edge">

This tag ensures the Internet Explorer uses the latest available rendering engine to render a web page, helping to ensure compatibility with older versions of IE. The http-equiv=X-UA-Compatible statement is used to degine an HTTP header for the document. X-UA-Compatible is a header to support compatibility with older versions of IE. The IE=edge statement instructs IE to use the highest mode available to render the web page. This ensures that the web page is rendered consistently across different versions of IE.

<title>

This tag shows text on the tab of the web page.

<body>

This tag contains all the elements that will be shown on the web page. The body holds the visuals to the web page.

Tags that link other files

<link rel="stylesheet" href="path-to-CSS-file-here">

This tag links CSS files to the document for use.

<script src="script.js">

This tag allows scripts to be loaded with the document.
